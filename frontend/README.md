# AI Tools Atlas - Frontend

Vue 3 + TypeScript + Vite + Tailwind CSS frontend for AI Tools Atlas.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs on port 5173 by default.

## Build

```bash
npm run build
```

## Configuration

Copy `.env.example` to `.env` and configure API endpoint:

```
VITE_API_BASE_URL=http://localhost:8080/api/v1
```

## Features

- Vue 3 with Composition API
- TypeScript for type safety
- Vite for fast development
- Tailwind CSS with RTL support
- Vue Router for navigation
- Pinia for state management
- Dark theme optimized for Arabic UI
